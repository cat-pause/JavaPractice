1.로그인 / 2.회원가입

1. 로그인 - 아이디, 비밀번호 중 뭐가 틀렸는지 로그출력
     ㄱ.관리자 로그인 (제품 목록 출력)
	1.상품 검색
		1-1.제품명 검색
		1-2.제조사 검색
	2.상품 추가
	3.상품 삭제
	4.재고 관리
		4-1.수량 증가
		4-2.수량 감소
		4-3.수량 설정
	5.로그아웃 
		- 로그인 화면으로 이동
	6.계정 관리
		6-1.계정 삭제
		6-2.특정 계정 소지금액 추가
     ㄴ.사용자 로그인 (제품 목록, 유저명, 소지금 출력)
 	1.금액 충전
	2.상품 주문
		- 제품번호 입력
		- 해당 제품 정보 출력
		2-1.주문하기
			- 제품 갯수 입력
			- 입력값과 재고 비교
			- 소지금으로 주문할수 있는지 확인
			- 소지금 감소
			- 사용자 로그인 화면으로 
		2-2.돌아가기
	3.로그아웃
		- 로그인 화면으로 이동

2. 회원가입
	- 아이디 입력
		- 중복 검사
	- 비밀번호 입력
	- 관리자 확인 
		- "admin"입력시 관리자 계정
	- 완료후 메인화면으로 이동(1.로그인/2.회원가입)
